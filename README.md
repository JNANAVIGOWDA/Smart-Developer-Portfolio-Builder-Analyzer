# Smart Developer Portfolio Builder & Analyzer

A web-based application that helps developers create, manage, and analyze professional portfolios. The system enables users to showcase their skills, projects, education, certifications, and experience through a structured portfolio interface.

---


## 📌 Features

- User Registration and Login
- Create and Manage Developer Portfolio
- Add and Edit Projects
- Manage Technical Skills
- Education Details Management
- Certifications Management
- Portfolio Analysis
- Resume PDF Download
- Responsive User Interface
- Secure Database Storage

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/JNANAVIGOWDA/Smart-Developer-Portfolio-Builder-Analyzer.git
```

2. Move the project to the XAMPP `htdocs` folder.

3. Start Apache and MySQL from XAMPP.

4. Import the SQL database into phpMyAdmin.

5. Open the project in your browser:

```
http://localhost/portfolio/
```
